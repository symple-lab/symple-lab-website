## 디자인 접근 1: 네오-브루탈리즘 기하학 (Neo-Brutalist Geometry)

**Design Movement**: 네오-브루탈리즘과 스위스 그래픽 디자인의 융합

**Core Principles**:
- 날카로운 각도와 대담한 기하학적 형태를 통한 시각적 충격
- 기능 우선주의: 모든 요소는 명확한 목적을 가짐
- 그리드 시스템의 의도적 파괴와 재구성
- 계층 구조의 극대화: 크기와 위치로 중요도 표현

**Color Philosophy**: 
- 주조색: 코발트 블루 (#0047AB) - 기술적 신뢰성과 혁신
- 보조색: 차콜 (#2C2C2C) - 무게감과 전문성
- 강조색: 실버 (#C0C0C0) - 금속적 정밀함
- 배경: 순백 (#FFFFFF)과 차콜의 극명한 대비
- 감정적 의도: 냉철함, 정확성, 미래지향성

**Layout Paradigm**: 
- 비대칭 모듈러 그리드: 화면을 불규칙한 기하학적 블록으로 분할
- 대각선 절단(diagonal cuts)과 각진 섹션 전환
- Z-패턴 시각 흐름을 활용한 정보 배치
- 고정된 사이드 네비게이션 바 (수직 차콜 스트립)

**Signature Elements**:
- 날카로운 삼각형 포인터와 화살표 모티프
- 두꺼운 테두리(4-6px)와 하드 섀도우
- 기하학적 패턴 배경 (육각형, 삼각형 그리드)
- 데이터 시각화를 위한 각진 차트와 다이어그램

**Interaction Philosophy**:
- 즉각적이고 명확한 피드백 (클릭 시 색상 반전)
- 호버 시 기하학적 형태의 확장/변형
- 스냅 애니메이션: 부드러움보다 정확성 우선

**Animation**:
- 선형 이징(linear easing) 또는 단계적 전환
- 요소 등장 시 슬라이드 + 페이드 (200-300ms)
- 페이지 전환 시 와이프 효과 (대각선 방향)
- 스크롤 시 패럴랙스 효과로 레이어 분리감 강조

**Typography System**:
- 헤드라인: Space Grotesk Bold (900) - 기하학적이고 기술적
- 본문: IBM Plex Sans Regular (400) - 가독성과 중립성
- 강조: Space Grotesk Medium (500)
- 계층: 크기 비율 1:1.5:2.5:4 (극단적 대비)
- 정렬: 주로 좌측 정렬, 제목은 대담한 크기로 시선 유도

## 디자인 접근 2: 테크-미니멀리즘 레이어링 (Tech-Minimalist Layering)

**Design Movement**: 일본 미니멀리즘과 하이테크 산업 디자인의 결합

**Core Principles**:
- 극도의 여백과 호흡감: 요소 간 넉넉한 공간 확보
- 레이어링을 통한 깊이감: 미묘한 고도 변화로 정보 구조화
- 정제된 우아함: 불필요한 장식 제거, 본질에 집중
- 그리드 기반 정렬: 모든 요소는 8px 그리드에 정확히 정렬

**Color Philosophy**:
- 주조색: 딥 코발트 (#003D82) - 차분하고 신뢰할 수 있는 기술
- 보조색: 슬레이트 그레이 (#3A3A3A) - 세련된 중립성
- 강조색: 플래티넘 실버 (#E5E5E5) - 빛나는 디테일
- 배경: 오프화이트 (#F8F9FA) - 눈의 피로 최소화
- 감정적 의도: 고요함, 집중, 프리미엄 품질

**Layout Paradigm**:
- 카드 기반 모듈 시스템: 각 정보 단위를 독립된 카드로 표현
- 플로팅 레이어: 요소들이 배경 위에 떠 있는 듯한 효과
- 비대칭 2-컬럼 레이아웃: 좌측 넓은 콘텐츠 영역, 우측 좁은 메타 정보
- 상단 고정 투명 네비게이션 (스크롤 시 블러 배경)

**Signature Elements**:
- 소프트 섀도우 (0 4px 20px rgba(0,0,0,0.08))
- 미세한 그라데이션 오버레이 (5% 투명도)
- 얇은 구분선 (1px, 10% 투명도)
- 원형 아바타와 둥근 모서리 (8-12px radius)

**Interaction Philosophy**:
- 부드럽고 예측 가능한 반응
- 호버 시 미묘한 상승 효과 (섀도우 증가 + 2px 이동)
- 포커스 상태는 얇은 코발트 링으로 표시
- 제스처 기반 네비게이션 힌트

**Animation**:
- 이징: ease-out (cubic-bezier(0.33, 1, 0.68, 1))
- 요소 등장: 페이드 + 상향 이동 (20px, 400ms)
- 페이지 전환: 크로스 페이드 (300ms)
- 스크롤 애니메이션: 스티키 헤더, 점진적 콘텐츠 노출

**Typography System**:
- 헤드라인: Outfit SemiBold (600) - 현대적이고 기하학적
- 본문: Inter Regular (400) - 최고의 화면 가독성
- 캡션: Inter Light (300)
- 계층: 크기 비율 1:1.25:1.75:3 (조화로운 스케일)
- 행간: 1.6-1.8 (넉넉한 호흡)
- 정렬: 좌측 정렬, 중앙 정렬은 히어로 섹션에만 사용

## 디자인 접근 3: 사이버네틱 다이나미즘 (Cybernetic Dynamism)

**Design Movement**: 사이버펑크 미학과 스위스 타이포그래피의 충돌

**Core Principles**:
- 에너지와 움직임: 정적인 페이지가 아닌 살아있는 인터페이스
- 기술적 복잡성의 시각화: 데이터와 연구의 역동성 표현
- 대비의 극대화: 밝음과 어두움, 정적과 동적의 충돌
- 정보 밀도: 많은 정보를 효율적으로 압축하여 전달

**Color Philosophy**:
- 주조색: 일렉트릭 블루 (#0066FF) - 에너지와 혁신
- 보조색: 매트 블랙 (#1A1A1A) - 깊이와 신비감
- 강조색: 네온 실버 (#B8C5D6) - 사이버네틱 하이라이트
- 액센트: 시안 (#00D9FF) - 디지털 펄스
- 배경: 다크 네이비 (#0A0E27) - 우주적 깊이
- 감정적 의도: 흥분, 미래, 첨단 기술

**Layout Paradigm**:
- 분할 화면 구성: 대각선 또는 곡선으로 화면 분할
- 오버레이 레이어: 정보가 여러 층위로 겹쳐짐
- 비선형 스크롤: 수평 + 수직 혼합 네비게이션
- 사이드바 없음: 전체 화면 몰입형 경험

**Signature Elements**:
- 글로우 효과 (box-shadow: 0 0 20px rgba(0,102,255,0.5))
- 애니메이션 그리드 배경 (움직이는 점 또는 선)
- 홀로그래픽 그라데이션 (블루-시안-퍼플)
- 데이터 스트림 시각화 (흐르는 숫자, 코드 라인)

**Interaction Philosophy**:
- 반응형 파티클 효과: 마우스 이동에 따라 배경 반응
- 호버 시 글로우 강화 + 색상 시프트
- 클릭 시 리플 애니메이션 (파동 효과)
- 스크롤 기반 스토리텔링: 내용이 단계적으로 드러남

**Animation**:
- 이징: ease-in-out (cubic-bezier(0.65, 0, 0.35, 1))
- 요소 등장: 스케일 + 페이드 (0.8 → 1, 500ms)
- 배경 애니메이션: 지속적인 미묘한 움직임 (무한 루프)
- 페이지 전환: 디졸브 + 줌 효과
- 텍스트 타이핑 효과: 중요 헤드라인에 적용

**Typography System**:
- 헤드라인: Orbitron Bold (700) - 사이버네틱 미래감
- 본문: Rajdhani Regular (400) - 기술적이면서 읽기 쉬움
- 코드/데이터: JetBrains Mono (400) - 모노스페이스 정밀함
- 계층: 크기 비율 1:1.4:2.2:3.5 (역동적 스케일)
- 문자 간격: 약간 넓게 (0.02-0.05em) - 디지털 느낌
- 정렬: 혼합 사용 (좌, 우, 중앙) - 역동성 강조
